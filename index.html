About Me

Hey! I'm Fil and I'm a FullStack Dev. What that means is that I have the ability to create websites and apps, all the way from start to finish. From the design work, to the front-end, the server side, and the database portion of the project. I graduated from the Full Stack Web Developer Course at General Assembly in Downtown Chicago in Dec 2018. Since then, i've been doing freelance programming work while I work a full time job. Eventually I'd love to get to a point where programming IS my full time job! I have experience in creating static and dynamic webpages using ReactJS. I prefer using the Express framework, which allows me to build a server side application using JavaScript and lets me build my REST API in JavaScript. I've used Ruby for my back end stuff, but I'd like to venture into using Python/Django for my back end. I like SQL databases. I use Postgresql as my database management system.

As for my paid freelance work. I've made; a website for a Speaker/Lighting company, a business website for an Investment Holding company. Basically an overdone portfolio that showcased his businesses, A business for a mom and pop store that sold candles, garden accents, florals and soaps. As for personal projects. I'm currently in the process of making a Dog Walking scheduling app that is similar to Wag!. In school I made a webapp called InkBerrie where tattoo artists can go there and upload pictures and other users who aren't artists can favorite pictures and artists. Feel free to hit me up whenever! I'm good at responding quickly. Can't wait to hear from ya! -Fil
